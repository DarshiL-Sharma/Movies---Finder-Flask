# 🎬 Flask Movie Finder App

A simple movie finder and management web application built with **Flask**. Users can add movies, upload/display images, and leave comments with ratings. This project focuses on clean data handling, user interaction, and basic full-stack development.

---

## 🚀 Features

* Add new movies with details
* Upload or attach movie images
* Add comments and ratings for movies
* Display movie list with images
* SQLite database integration
* Form validation using Flask-WTF

---

## 🛠️ Tech Stack

* Python
* Flask
* Flask-Bootstrap
* Flask-WTF
* Flask-SQLAlchemy
* SQLite

---

## 📂 Functionality Overview

* **Home Page:** Displays all movies with images and details
* **Add Movie:** Users can add a new movie using a form
* **Comments Section:** Users can leave comments and ratings on movies
* **Image Support:** Each movie includes an image (via URL or uploaded file)

## ▶️ How to Run

1. Clone the repository:

```bash id="clone2"
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies:

```bash id="install2"
pip install flask flask-bootstrap flask-wtf flask-sqlalchemy
```

3. Run the app:

```bash id="run2"
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000/
```
